# MapSyncer 功能测试文档

本文档用于记录 MapSyncer for XaeroWorldmap 模组的功能测试结果。

**测试环境要求**：
- Minecraft 1.21.X + NeoForge
- Xaero's World Map mod（最低版本 1.37.0）
- 服务端和客户端均安装 MapSyncer
